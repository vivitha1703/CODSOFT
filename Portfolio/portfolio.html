Classified Website

The website is designed to facilitate the buying and selling of items through classified advertisements. Leveraging HTML and CSS for the front-end, the user interface provides an intuitive and visually appealing platform. Flask, a lightweight web framework for Python, serves as the backend to handle user interactions and data processing. Travel Planning Web App

This website serves as a comprehensive guide for travelers exploring the diverse attractions within Tamilnadu. Developed using HTML, CSS, and the Flask framework, the site offers an engaging and informative user interface. Visitors can access detailed information about various destinations, including historical landmarks, cultural hotspots, natural wonders, and more. The use of HTML and CSS ensures a visually appealing layout, with interactive features that enhance the user experience. Flask, functioning as the backend framework, manages data retrieval and presentation, allowing users to navigate seamlessly through the site. The platform includes features such as search functionality, curated itineraries, user reviews, and real-time updates on events and festivals. By combining these technologies, the website aims to inspire and assist travelers in planning memorable journeys throughout the vibrant landscapes of Tamilnadu.

Lung Cancer Diagnosing

This website employs machine learning, Python Flask, HTML, CSS, and Docker to predict lung cancer based on user-input data. It features an intuitive HTML/CSS interface, with Flask handling data processing and machine learning. The entire system is containerized in Docker, ensuring portability and scalability. The website serves as a valuable tool for early lung cancer risk prediction, bridging healthcare technology and web development. Users can input relevant information, and the system, running in a Dockerized environment, processes the data using machine learning algorithms implemented in Python Flask. The front-end, built with HTML and CSS, provides an intuitive and user-friendly interface for inputting data and viewing predictions.
